# Toasty Task Roadmap

Toasty Task is an actively developed open-source task manager. The roadmap is
intentionally small and focused on making the existing product easier to run,
review, and contribute to.

## Current Priorities

- Harden authentication and multi-tenant authorization boundaries.
- Expand automated tests for task mutations, notes, recurrence, and scoring.
- Improve setup documentation for local PostgreSQL and Clerk configuration.
- Publish sanitized product screenshots and a public demo environment.
- Continue responsive web and mobile client work.

## Near-Term Releases

### v0.1

- Publish the initial open-source release.
- Document setup, contribution, and security-reporting workflows.
- Track public bugs and feature requests through GitHub Issues.

### v0.2

- Add broader API and repository test coverage.
- Improve onboarding and sample-data tooling.
- Document architecture decisions for scoring and offline mobile sync.

### Future

- Expand maintainer automation for issue triage, pull request review, release
  notes, and regression testing.
- Evaluate import and export workflows for common task-management formats.

## Proposing Work

Open a GitHub issue to propose a feature or report a bug. Keep private data,
credentials, and security reports out of public issues.
